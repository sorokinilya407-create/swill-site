Telegram — полный гайд

Почему не работает: РКН блокирует WebSocket веб-версии. Решение — MTProto прокси.

Способ 1: Готовый прокси (30 секунд)

  1. Скачай файл с прокси по ссылке ниже
  2. Открой Telegram → Настройки → Данные и память → Прокси
  3. Жми «Добавить прокси» → MTProto
  4. Скопируй данные из файла и вставь
  5. Готово — Telegram работает без VPN
SoliSpirit/mtproto — свежие прокси каждые 12ч

Способ 2: Свой прокси (для гиков)

Подними на VPS за 5 минут — будет только твой и без лимитов.

# Установка Docker
curl -fsSL https://get.docker.com | sh
# Запуск MTProto с маскировкой под Google
docker run -d --restart=unless-stopped --name mtg -p 443:3128 nineseconds/mtg:2 run https://www.google.com
# Получить ссылку
docker logs mtg 2>&1 | grep "tg://"

YouTube — убираем тормоза

Ситуация: РКН режет скорость YouTube через DPI. Решается утилитами GoodbyeDPI/Zapret.

GoodbyeDPI (Windows 10/11)

  1. Скачай последнюю версию
  2. Распакуй в C:\goodbyedpi\ (без русских букв!)
  3. Запусти 1_russia_blacklist_dnsredir.cmd от имени Администратора
  4. Не закрывай окно — YouTube полетит на полной скорости
