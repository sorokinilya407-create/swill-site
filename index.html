Telegram — пошаговый гайд

Суть проблемы: РКН режет веб-версию. Решение — встроенный прокси MTProto. Это как секретный туннель внутри Telegram. Настраивается за 1 минуту.

Способ для всех: готовый прокси

Ниже — ссылки на файлы с рабочими прокси. Обновляются автоматически.

SoliSpirit/mtproto — свежие прокси каждые 12 часов
rjsxrd/mtproto — резервный список раз в 2 дня

Как добавить прокси (открой Telegram и делай):

  1. Нажми Настройки (шестерёнка) → Данные и память → Прокси
  2. Нажми «Добавить прокси» (плюсик или кнопка внизу)
  3. Выбери тип MTPROTO (это важно!)
  4. Теперь смотри в файл с прокси (ссылки выше). Там строка вида:
    tg://proxy?server=123.456.78.90&port=443&secret=ee1234567890abcdef...
  5. Вот что куда вставлять:
    • Hostname — это то, что после server= (например 123.456.78.90)
    • Port — это после port= (обычно 443)
    • Secret — это после secret= (вся длинная строка, копируй целиком)
  6. Нажми Сохранить
  7. Выбери этот прокси в списке и включи переключатель «Использовать прокси»
Лайфхак для телефона: Просто открой ссылку tg://proxy?... в браузере — Telegram сам предложит добавить прокси. Копируй всю строку из файла и вставляй в адресную строку.

Хочу свой прокси (для продвинутых)

Подними на своём VPS — будет только твой, без лимитов и чужих глаз.

# Установи Docker (если нет)
curl -fsSL https://get.docker.com | sh

# Запусти MTProto с маскировкой под Google
docker run -d --restart=unless-stopped --name mtg -p 443:3128 nineseconds/mtg:2 run https://www.google.com

# Получи свою ссылку для подключения
docker logs mtg 2>&1 | grep "tg://"

YouTube — убираем тормоза

Ситуация: РКН режет скорость YouTube через DPI. Решается утилитами GoodbyeDPI/Zapret.

GoodbyeDPI (Windows 10/11)

  1. Скачай последнюю версию
  2. Распакуй в C:\goodbyedpi\ (без русских букв!)
  3. Запусти 1_russia_blacklist_dnsredir.cmd от имени Администратора
  4. Не закрывай окно — YouTube полетит на полной скорости
